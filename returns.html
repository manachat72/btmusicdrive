ที่ btmusicdrive เรามุ่งมั่นที่จะมอบประสบการณ์การฟังเพลงที่ดีที่สุด "เสียบปุ๊บ ฟังปั๊บ" หากสินค้ามีปัญหา เรายินดีดูแลท่านตามเงื่อนไขการรับประกันดังต่อไปนี้ เพื่อความมั่นใจในการสั่งซื้อ

1. การรับเปลี่ยนและคืนสินค้า

ท่านสามารถแจ้งขอเปลี่ยนหรือคืนสินค้าได้ภายใน 7 วัน นับจากวันที่ได้รับสินค้า (อ้างอิงจากสถานะการจัดส่งสำเร็จของบริษัทขนส่ง) โดยครอบคลุมกรณีดังต่อไปนี้:

  • สินค้าชำรุดเสียหาย: แฟลชไดร์ฟเสีย, แตกหักตั้งแต่ได้รับ, หรือคอมพิวเตอร์/เครื่องเสียงไม่สามารถอ่านข้อมูลได้ตั้งแต่ครั้งแรกที่ใช้งาน
  • ได้รับสินค้าไม่ถูกต้อง: ได้รับแนวเพลงไม่ตรงตามที่สั่งซื้อ หรือได้รับสินค้าผิดรุ่น/ผิดความจุ
  • ไฟล์เพลงมีปัญหา: ไฟล์สะดุด, เพลงขาดหาย, หรือไฟล์เสียจนไม่สามารถรับฟังได้อย่างสมบูรณ์

2. เงื่อนไขที่ไม่อยู่ในการรับประกัน (ไม่รับคืน)

เนื่องจากสินค้าเป็นสื่อบันเทิงและข้อมูลดิจิทัลที่สามารถคัดลอกได้ ทางร้านขอสงวนสิทธิ์ ไม่รับเปลี่ยนหรือคืน ในกรณีดังต่อไปนี้:

  • กรณีเปลี่ยนใจ: ลูกค้าเปลี่ยนใจไม่ต้องการสินค้าแล้วหลังจากได้รับสินค้า, สั่งซื้อผิดรุ่นด้วยตนเอง หรือไม่ถูกใจแนวเพลงหลังจากที่ได้ทดลองฟัง
  • ความเสียหายจากผู้ใช้: ทำสินค้าตกหล่น, แตกหัก, โดนน้ำ, ดัดแปลง, นำไปใช้งานผิดประเภท หรือมีการ Format แฟลชไดร์ฟ / ลบ-เพิ่มเพลงด้วยตนเองจนระบบข้อมูลเสียหาย
  • ปัญหาจากอุปกรณ์ของลูกค้า: เครื่องเสียงรถยนต์รุ่นเก่ามากที่ไม่รองรับไฟล์ MP3, ไม่รองรับความจุของแฟลชไดร์ฟ, หรือไม่รองรับระบบไฟล์ (เช่น รถยนต์บางรุ่นต้องใช้ FAT32 เท่านั้น) *หากไม่แน่ใจ แนะนำให้ทักแชทสอบถามรุ่นรถกับแอดมินก่อนสั่งซื้อครับ

3. หลักฐานที่ต้องใช้ในการขอเคลมสินค้า

เพื่อความรวดเร็วในการดำเนินการและเป็นการรักษาสิทธิ์ของท่าน รบกวนลูกค้าเตรียมหลักฐานดังต่อไปนี้:

กรุณาถ่ายวิดีโอเป็นหลักฐานเสมอ

  • วิดีโอขณะแกะกล่องพัสดุ: ถ่ายให้เห็นใบปะหน้าชัดเจนและสภาพกล่อง/สินค้าตั้งแต่เริ่มแกะ
  • วิดีโอขณะทดสอบใช้งาน: ถ่ายให้เห็นตอนเสียบใช้งาน และแสดงปัญหาที่พบ (เช่น หน้าจอเครื่องเสียงขึ้นเตือน Check USB, No File หรือเล่นแล้วสะดุด)

  3. ส่งคืนสินค้า: หากเข้าเงื่อนไขการเคลม แอดมินจะแจ้งที่อยู่สำหรับจัดส่งคืน (สินค้าต้องกลับมาในสภาพสมบูรณ์ พร้อมกล่องบรรจุภัณฑ์)
